## Game Development Context

Arcade World is a web-based gaming platform featuring:
- Multiple arcade games with unified progression
- Virtual currency system (points → tickets → prizes)
- Themed arcade environments with progressive unlocking
- Social features including multiplayer, tournaments, and leaderboards
- Mobile-optimized gaming experience

The technical stack is optimized for real-time gaming, player progression tracking, and cross-device compatibility using Next.js, TypeScript, Firestore, and custom game engine components.
